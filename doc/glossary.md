# 用語集

## 1.名付け規則

英語と日本語を併用。日本語(ローマ字)を使用する場合、「かなの省略」の発生する記述方式は可読性が極端に落ちるのでNG
ex.様式区分
×yoshikikbn
〇yohsikiKbn
〇youhshikiKbn
